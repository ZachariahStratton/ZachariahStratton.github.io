About

I have extensive experience in modifying the source code of various video games as well as datamining for information. The most popular of these are my various modifications to the popular competitve game Super Smash Brothers Melee, which have a combined total downloads of over 33,000. They are also featured in the 20XX Hack Pack which has hundreds of thousands of downloads and has been featured in many professional tournaments, as well as the Netplay Community Build which is widely used for online training.

My Work

A majority of the modifications I created were changes to textures and animations within the game to allow greater customization for players. This required explicit documentation to assist fellow users working on the same goal as well as direct modification of machine code in many situations.
