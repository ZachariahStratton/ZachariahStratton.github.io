G-Bib
About

G-Bib is a program designed to allow market users to utilize a .bib library to provide citations (in-line and reference page) to a multi-user environment. We focused on meeting the following objectives:

  • The application will function entirely as a free Google Docs plugin
  • Edits to document will match IEEE citation formatting
  • The application uses bibtex formatting and .bib libraries
  • The application will be accessible to .bib experts and novices
  • The application will not overwrite or lose data when multiple users are utilizing it and will respect privacy of the users
My Role

For development in this project I took on the role of Software Lead for Data Storage and Management. My main tasks were working with Google Apps Script to create efficient and fast storage and retrieval of files needed by the program, mainly the user's .bib library as well as a .JSON formatted file used by our program.

I also worked in a team environment involving strict weekly schedules, meetings, progress reports and deadlines. Our team successfully met all planned goals with our project on time.
